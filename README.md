## ðŸŽ¨ Customization

### Colors
Edit CSS variables in `css/style.css` (lines 28-60):
- `--primary`: Main accent color (default: #6366f1)
- `--secondary`: Secondary accent (default: #8b5cf6)
- `--bg`: Background color
- `--text`: Text color

### Fonts
Currently using:
- **Headings**: Poppins (from Google Fonts)
- **Body**: Lora (from Google Fonts)

Change in `index.html` (Google Fonts link) and `css/style.css` (`--font-heading` and `--font-body` variables).

### Images

- Hero background: Line ~85
- About portrait: Line ~97
- Gallery images: Lines ~127-215
